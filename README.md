# 🎬 Movie Explorer

A modern Movie & TV Show search application built using HTML, CSS, JavaScript, and the TVMaze API.

## 🚀 Features

- Search Movies and TV Shows
- Multiple Search Results
- Movie Posters
- Ratings & Genres
- Language & Status Information
- Premiered Date
- Search History
- Favorites System ❤️
- Dark / Light Theme 🌙
- Loading Animation
- Responsive Design
- Enter Key Search Support
- Local Storage Support

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- TVMaze API
- Local Storage

## 🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/harviesharma05-tech/movie-explorer.git
```

2. Open the project folder

3. Run:

```txt
index.html
```

in your browser.

## 🎯 How to Use

1. Enter a movie or TV show name.
2. Click Search.
3. Browse search results.
4. Add favorites.
5. View search history.
6. Toggle Dark/Light mode.

## 📡 API Used

TVMaze API

https://api.tvmaze.com

No API key required.

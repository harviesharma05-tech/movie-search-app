# Movie Search App 🎬

A movie search application built using HTML, CSS, JavaScript, and OMDb API.

## Features

- Search any movie
- Movie poster display
- Genre information
- IMDB rating
- Plot summary
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- OMDb API

## How to Run

1. Get an OMDb API key.
2. Replace YOUR_API_KEY in script.js.
3. Open index.html.

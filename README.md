# Movie Search App 🎬

A movie/show search application built using HTML, CSS, JavaScript and TVMaze API.

## Features

- Search TV shows and movies
- Display poster
- Show rating
- Show genres
- Display summary
- No API key required

## Technologies Used

- HTML
- CSS
- JavaScript
- TVMaze API

## How to Run

1. Download project
2. Open index.html
3. Search any movie/show
